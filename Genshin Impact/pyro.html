PYRO (Fire)

Pyro is an Element out of the Seven Elements in Genshin Impact, also known as Fire.

The element is able to interact with other elements and other elements are able to interact with it as well.

Characters that use Pyro:


  • Diluc
  • Klee
  • Amber
  • Xiangling
  • Bennett

Different Elemental Combos with Pyro:

Element Result
Cryo Melt
Hydro Vaporize
Electro Overload
Ameno Swirl
Geo Crystallize
Dendro Burning
